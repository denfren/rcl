# rcl format

## Synopsis

    rcl format <input>

Shorthands:

    rcl fmt
    rcl f

## Description

Read an RCL expression from the file `<input>`, and format it according to the
standard style. When `<input>` is `-`, read from stdin instead. Print the result
to stdout.
